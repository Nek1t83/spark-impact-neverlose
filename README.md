# Spark Impact (Neverlose CS:GO)

Lua-скрипт для чита **Neverlose** в CS:GO. Рисует **2D-эффект искр** в месте
попадания пули — вспышка в центре + разлёт лучей с физикой (гравитация,
затухание, отскок от поверхностей, 3D-перспектива). Эффект полностью
локальный, его видишь только ты.

---

## Что в репозитории

```
spark_impact.lua    — сам скрипт (всё в одном файле, без внешних ресурсов)
```

Ни звуков, ни картинок не требуется — всё рисуется через `render.line`
и `render.circle`.

---

## Настройки в меню

| Настройка                | Что делает                                                                |
|--------------------------|---------------------------------------------------------------------------|
| **Включить эффект**      | Главный переключатель.                                                    |
| **Только мои выстрелы**  | Если выкл — искры от попаданий любых игроков (видно только тебе).         |
| **Цвет искр**            | Основной цвет (RGBA). Альфа применяется как множитель к затуханию.        |
| **Количество искр**      | 8..80 искр на одно попадание. Жёсткий потолок в коде — 600 одновременно.  |
| **Разлёт**               | Начальная скорость искр (units/sec).                                      |
| **Гравитация**           | Сила тяги вниз. 0 = искры летят прямо.                                    |
| **Длительность**         | Сколько живёт искра, в секундах × 10 (т.е. 9 = 0.9с).                      |
| **Длина искр**           | Длина хвоста, секунды × 100. Чем больше — тем «длиннее» полоска.           |
| **Толщина искр**         | 1..6 пикселей. Реальная толщина ещё корректируется 3D-перспективой.       |
| **Вспышка в центре**     | Короткий жёлтый кружок + белый кор в первые 0.06с.                        |
| **Размер вспышки**       | Радиус вспышки в пикселях.                                                |
| **Градиент (белый центр)** | Искра светлее в середине (имитирует раскалённый кончик).                |
| **3D-перспектива**       | Толщина искры зависит от расстояния до камеры.                            |
| **Отскок от поверхностей** | Искры отражаются от стен/пола через `utils.trace_line`.                 |
| **Сила отскока**         | 0..80%. 0 = поглощение, 80% = почти упругий отскок.                       |

---

## Производительность

- Жёсткий лимит в **600 искр** одновременно — больше не спавнится, пока
  старые не умрут.
- Трассировка для отскоков делается только для искр со скоростью > 30 unit/s
  (медленные и так уже на полу).
- Все «горячие» функции (`math.sin`, `render.line`, …) залокалены в начале
  файла, никаких глобал-лукапов в цикле рендера.
- При лагах / переключении окна `dt` обрезается до 0.1с, чтобы искры не
  телепортировались сквозь стены.

На обычном железе спокойно тянет даже на максимальных настройках
(80 искр × частые попадания из автомата).
